# База Проектов

> [!info] Автоматическая база данных
> Все проекты из папки `02_Проекты` автоматически отображаются здесь в табличном виде.

---

## Все проекты

```dataview
TABLE
  status as "Статус",
  priority as "Приоритет",
  area as "Область",
  start as "Начало",
  due as "Дедлайн",
  round(length(filter(file.tasks, (t) => t.completed)) / max(length(file.tasks), 1) * 100) + "%" as "Прогресс"
FROM "02_Проекты"
WHERE type = "project"
SORT choice(status = "active", 1, choice(status = "on-hold", 2, 3)) ASC, priority ASC, due ASC
```

## Фильтры

### Активные проекты

```dataview
TABLE WITHOUT ID
  file.link as "Проект",
  priority as "Приоритет",
  due as "Дедлайн",
  length(filter(file.tasks, (t) => !t.completed)) as "Задач осталось"
FROM "02_Проекты"
WHERE type = "project" AND status = "active"
SORT priority ASC, due ASC
```

### На паузе

```dataview
LIST
FROM "02_Проекты"
WHERE type = "project" AND status = "on-hold"
SORT file.mtime DESC
```

### Завершенные

```dataview
TABLE WITHOUT ID
  file.link as "Проект",
  start as "Начало",
  due as "Завершено"
FROM "02_Проекты" OR "05_Архив"
WHERE type = "project" AND status = "completed"
SORT due DESC
LIMIT 10
```

## Статистика проектов

```dataview
TABLE WITHOUT ID
  status as "Статус",
  length(rows) as "Количество"
FROM "02_Проекты"
WHERE type = "project"
GROUP BY status
```

### По областям ответственности

```dataview
TABLE WITHOUT ID
  area as "Область",
  length(rows) as "Проектов",
  length(filter(rows, (r) => r.status = "active")) as "Активных"
FROM "02_Проекты"
WHERE type = "project" AND area
GROUP BY area
SORT length(rows) DESC
```
