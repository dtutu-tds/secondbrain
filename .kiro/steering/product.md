# Product Overview

This is an Obsidian "Second Brain" system - a personal knowledge management and productivity framework. The system helps users manage daily journaling (Lite/Full modes), projects, tasks, biohacking logs, time tracking, and weekly reviews.

## Core Purpose

Enable users to:
- Offload mental burden through structured note-taking
- Track progress with quantifiable metrics
- Focus on 1-3 concrete daily actions
- Maintain work-life balance with 20-minute morning/evening routines

## Key Principles

- **Law of Three**: Focus on 3 key tasks (A1, A2, A3)
- **80/20 Rule**: Identify high-impact activities
- **One Main Task (A1)**: Single most important daily task
- **Reflective Thinking**: Conscious reflection through writing

## Target User

Works on both PC and mobile, doesn't use external task managers, practices biohacking, wants to see measurable progress and maintain focus without system overhead.
