# Language Preferences

## Communication Language

**Always respond in Russian** when interacting with the user.

## Documentation Language

- User-facing documentation: Russian
- Technical specifications: Russian
- Code comments: English (standard practice)
- Variable/function names: English (standard practice)

## File Content Language

- Markdown content in notes: Russian
- YAML frontmatter keys: English (for plugin compatibility)
- Template instructions: Russian
- Dashboard labels: Russian with emoji where appropriate
